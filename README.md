# きもちちゃん / Kimochi-chan

気持ちを話しかけると、表情と声で返事してくれるブラウザキャラクターアプリです。

🔗 **公開URL**: https://keikei93.github.io/kimoticyan/kimoticyan.html

---

## 概要

テキスト入力（PC）またはボタン選択（iPhone）でキーワードを送ると、きもちちゃんが7種類の表情と音声で反応します。ページ下部の地面をてくてく歩き回り、時々転んだり、画面端からこっそり覗いたりします。

---

## 機能

### キャラクター
- SVG描画によるオリジナルキャラクター
- 表情7種類：normal / happy / shy / troubled / sleepy / angry / blank
- バウンスアニメーション（返答時）
- てくてく歩き・転倒・のぞきアニメーション

### 入力
| 環境 | 方式 |
|------|------|
| PC・ブラウザ | テキスト入力（UA自動判定） |
| iPhone・iPad | ボタン選択（6ボタン） |

### ボタン仕様（iOS）
- 各ボタンは1語表示、押すたびにカテゴリ内でランダムに切り替わる
- 挨拶ボタンのみ時間帯で自動切り替え（朝/昼/夜/深夜）

| ボタン | ラベル例 | 対応感情 |
|--------|---------|---------|
| あいさつ | おはよう / こんにちは / こんばんは / おやすみ | happy / normal / sleepy |
| うれしい系 | うれしい / たのしい / わくわく / さいこう / やった | happy |
| つかれた系 | つかれた / しんどい / だるい / へとへと | troubled |
| ありがとう系 | ありがとう / すき / かわいい / だいすき | shy |
| むかつく系 | むかつく / いらいら / やばい / わからない / さみしい | angry |
| こわい系 | こわい / びっくり / おどろいた / やばい | blank |

### 音声
- Web Speech API（ブラウザ標準機能・外部API不使用）
- 日本語音声：rate 1.0 / pitch 1.6
- 音声オン/オフ切り替えボタンあり
- iOS対応：初回タップ時にウォームアップ処理あり（冒頭途切れ防止）

### 返答
- キーワードマッチング方式（15カテゴリ）
- 各カテゴリ14フレーズ、ランダム選択
- マッチしない場合はデフォルト返答（16フレーズ）

## ローカルでの実行

```bash
git clone https://github.com/keikei93/kimoticyan.git
cd kimoticyan
# kimoticyan.html をブラウザで開く
```

音声機能はローカルでも動作します（ブラウザのWeb Speech API対応が必要）。

---

## 開発メモ

- iOSのキーボード問題（入力時にキャラが画面外へ押し上げられる）はボタン方式UIへの切り替えで解決
- Web Speech APIはiOSで初回タップ後にウォームアップ発声が必要
- てくてくアニメはCSS `steps(1)` によるコマ送りで実装
- 転倒アニメの回転軸はキャラ右端・足底に設定
